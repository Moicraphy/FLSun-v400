English:
1.backup and save a copy of your V400 files that were loaded into cura from the FLSUN thumb drive and then remove these files from cura (or start with a clean install)

2. Open cura and select Help-->Show Configuration Folder

3. Place the "flsun_v400_extruder" folder in the "extruders" directory under the cura configuration path.

4. Place the "flsun_v400_intent" folder in the "intent" directory under the cura configuration path.

5. Place the "flsun_v400_quality" folder in the "quality" directory under the cura configuration path.

6. Place the "flsun_v400_variants" folder in the "variants" directory under the cura configuration path. (NOTE: only copy the nozzle sizes you want to use if you dont want all nozzle sizes to be shown)

7. Choose either "flsun_v400_STOCK.def.json" or "flsun_v400_updated_firmware.def.json" depending on if you have updated the stock firmware or not and RENAME TO flsun_v400.def.json!!! place the chosen file in the "definitions" directory in the configuration path.

8. Place the "flsun_v400. stl" file under the "meshes" directory in the configuration path.

9. Close all open CURA software and add V400 after opening it again.

V1.0.1 There are three nozzle sizes (0.25, 0.4, 0.6, and 0.8) to select from as well as under the 0.4 nozzle selection you have sub catagories of "default" "engineering" "visual" and "draft".

I also reworked the definition file for the V400 in order to remove the annoying yellow "warning" that occurs when you set print speed to 400

I have added testing modles for those that want to print with a standardized test print to help give feedback on these profiles.

Deutsch:
1. Sichern und speichern Sie eine Kopie Ihrer V400-Dateien, die vom FLSUN-Stick in Cura geladen wurden, und entfernen Sie diese Dateien dann aus Cura (oder beginnen Sie mit einer Neuinstallation).

2. Öffnen Sie Cura und wählen Sie Hilfe-->Konfigurationsordner anzeigen

3. Platzieren Sie den Ordner „flsun_v400_extruder“ im Verzeichnis „extruders“ unter dem Cura-Konfigurationspfad.

4. Platzieren Sie den Ordner „flsun_v400_intent“ im Verzeichnis „intent“ unter dem Cura-Konfigurationspfad.

5. Platzieren Sie den Ordner „flsun_v400_quality“ im Verzeichnis „quality“ unter dem Cura-Konfigurationspfad.

6. Platzieren Sie den Ordner „flsun_v400_variants“ im Verzeichnis „variants“ unter dem Cura-Konfigurationspfad. (HINWEIS: Kopieren Sie nur die Düsengrößen, die Sie verwenden möchten, wenn Sie nicht möchten, dass alle Düsengrößen angezeigt werden.)

7. Wählen Sie entweder „flsun_v400_STOCK.def.json“ oder „flsun_v400_updated_firmware.def.json“, je nachdem, ob Sie die Stock-Firmware aktualisiert haben oder nicht, und RENAME TO flsun_v400.def.json!!! Platzieren Sie die ausgewählte Datei im Verzeichnis "definitions" im Konfigurationspfad.

8. Platzieren Sie die Datei „flsun_v400.stl“ im Verzeichnis „meshes“ im Konfigurationspfad.

9. Schließen Sie alle geöffnete CURA-Software und fügen Sie V400 nach dem erneuten Öffnen hinzu.

V1.0.1 Es stehen drei Düsengrößen (0,25, 0,4, 0,6 und 0,8) zur Auswahl, außerdem gibt es unter der 0,4-Düsenauswahl Unterkategorien von „Standard“, „Technik“, „Visuell“ und „Entwurf“.

Ich habe auch die Definitionsdatei für den V400 überarbeitet, um die lästige gelbe "Warnung" zu entfernen, die auftritt, wenn Sie die Druckgeschwindigkeit auf 400 einstellen

Ich habe Testmodelle für diejenigen hinzugefügt, die mit einem standardisierten Testdruck drucken möchten, um Feedback zu diesen Profilen zu geben.
